# Desenvolvimento de Software em Python: Sistema Modular e Programação Defensiva

Este repositorio apresenta o desenvolvimento de um sistema modular construido em Python, focado em engenharia de software aplicada, tratamento de excecoes e controle de fluxos com validacoes estruturadas.

## Tecnologias e Conceitos Aplicados

* **Modularidade de Software:** Encapsulamento logico em funcoes dedicadas isolando responsabilidades.
* **Programacao Defensiva:** Estruturas de controle de excecoes try/except com o tipo especifico ValueError, mitigando falhas por entradas de dados invalidas.
* **Gerenciamento de Fluxo:** Controle continuo do ciclo de execucao do aplicativo utilizando lacos logicos encadeados com interrupcoes controladas por meio de estruturas while True e break.
* **Nomenclatura Internacional:** Codigo estruturado adotando as melhores praticas do mercado, com variaveis e metodos em ingles para maior manutenibilidade.
